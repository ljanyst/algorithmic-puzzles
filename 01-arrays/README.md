
Arrays
======

Find a pair in an array with a given sum
----------------------------------------

Input:

    10
    2 5 6 7 8 3 1 2 3 8
    13

 * line 1 - number of elements in the erray
 * line 2 - the array
 * line 3 - the sum

Output:

    n m

 * the indexes of the element summing up to `sum` of `-1 -1` if no pair is found
